'UNi Portal System'
1. Install Git from git-scm.com
2. Make account on Github.com
3. Initialize Git with your ID on Github.com by using following steps.
    3.1 git config --global user.email = "Youremail@example.com"
    3.2 git config --global user.name = "YourIDName"
4. Then goto the folder where you want to store this project.
5. Open GitBash and type 'git clone https://github.com/Haroon99/UNi-Portal-Management-System.git'
    - It will clone this project to your directory.
6. Make changes in the project. Then open GitBash again by right clicking on the empty area of the folder and select "Open GitBash here".
7. Write following commands in sequences.
    7.1 'git add .' -> 'git commit -m "any message(like add filename / update something)"' -> 'git push'
    - It will save your changes on the Github.

For Commands Reference Visit: https://confluence.atlassian.com/bitbucketserver/basic-git-commands-776639767.html
